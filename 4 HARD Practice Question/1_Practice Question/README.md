# SQL Practice Question

**Q: Show all of the patients grouped into weight groups.**

Show the total amount of patients in each weight group.

Order the list by the weight group descending.

For example:

- 100–109 → 100 weight group
- 110–119 → 110 weight group
- 120–129 → 120 weight group

## Expected Output

| patients_in_group | weight_group |
|------------------|-------------|
| 6 | 140 |
| 59 | 130 |
| 191 | 120 |
| 426 | 110 |
| 507 | 100 |
| 403 | 90 |
